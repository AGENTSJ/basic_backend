Instructions

Endpoints are give bellow use postman or other api testers to use these end points

Take care of few steps before using the end points

  1. Initialize mongodb server with 'mongod' command
  2. dont forget to provide body part in your request example:
    { "name":"Jhon Doe", "password":"tyg567JkR" }
  3. Before using '/login' and '/auth' insert few values into the data base with '/createuser' provide body like above mentioned example
  4. '/auth' endpoint takes Authentificationtoken in header 'auth-token':'token aquired from /login'

Endpoints

  • To create user : Request type :Post http://localhost:2400/createuser
  • To login with Authtoken: Request type :Get http://localhost:2400/login
  • To get Authentification token : Request type :Get http://localhost:2400/auth
